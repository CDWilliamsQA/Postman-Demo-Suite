# Security Policy

## Supported Versions

This project is a demonstration framework and is maintained for educational and portfolio purposes.

---

## Reporting a Vulnerability

If you discover a security issue related to:

- Environment configurations
- Test data exposure
- CI pipeline misuse

Please report it directly to the project maintainer.

Do not open a public issue for security concerns.

---

## Scope

This repository does not handle production credentials or sensitive data. All APIs used are public demonstration endpoints.
