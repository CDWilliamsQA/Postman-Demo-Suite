# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, respectful, and harassment-free environment for everyone, regardless of background, identity, or experience level.

We value professionalism, constructive communication, and collaboration.

---

## Expected Behaviour

Participants in this project are expected to:

- Be respectful and considerate in communication
- Provide constructive feedback
- Assume positive intent
- Focus on improving the project
- Support an inclusive and welcoming environment

---

## Unacceptable Behaviour

The following behaviours are not tolerated:

- Harassment, discrimination, or abusive language
- Personal attacks or derogatory remarks
- Intimidation or threats
- Publishing private information without consent
- Disruptive or unprofessional conduct

---

## Enforcement

Project maintainers reserve the right to remove or reject contributions that violate this Code of Conduct.

---

## Reporting Issues

If you experience or witness behaviour that violates this Code of Conduct, please contact the maintainer directly.

All reports will be handled with discretion.

---

This Code of Conduct applies to all project spaces, including GitHub issues, pull requests, and discussions.
